# Nutriflow 🍎

O **Nutriflow** é uma ferramenta de gestão nutricional focada no controle de dietas e planos alimentares, utilizando práticas modernas de desenvolvimento e testes.

## 🚀 Tecnologias

* **Node.js**: Runtime Javascript.
* **TypeScript**: Tipagem estática para maior manutenibilidade.
* **Jest**: Framework de testes para garantir a integridade das regras de negócio.
* **Fastify**: Framework web ágil e eficiente.
